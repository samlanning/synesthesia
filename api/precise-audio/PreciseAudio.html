    Class PreciseAudio

    An audio player that can seek and provide timestamps with millisecond accuracy. In contrast to the <audio> tag, this class will load an entire track into memory as a raw waveform, as otherwise, with most codecs, it's impossible to seek to accurate locations in songs.

    ExampleUsage:

    import PreciseAudio from '@synesthesia-project/precise-audio';

const audio = new PreciseAudio();
audio.loadAudioFile(...);

    Motivation, more usage instructions, and other details can be found in the project GitHub repository

    Package: @synesthesia-project/precise-audio
